Welcome to SparkUp!

This tool will help you install Apache Spark on your Windows machine in a few simple steps.

Installation

Execute the following in PowerShell:

Invoke-Expression ([System.Text.Encoding]::UTF8.GetString((Invoke-WebRequest -Uri https://sparkup.github.io/sparkup.ps1 -UseBasicParsing).Content))

After the script has run, restart your PowerShell session or start a new one to ensure all changes take effect.

Requirements

The script checks for the following prerequisites:

  • Java installed and available in the PATH.
  • Python installed and available in the PATH.

What does the script do?

The `sparkup.ps1` script performs the following actions:

  • Downloads Apache Spark to your user directory.
  • Updates the PYTHONPATH to include paths for PySpark and Py4J.
  • Downloads the necessary winutils.exe for Hadoop compatibility on Windows.
  • Sets up SPARK_HOME and HADOOP_HOME environment variables.

Getting Started with Spark

Once the installation completes, you can start using Apache Spark! Try out the spark-shell or pyspark commands in your terminal to launch the Spark interactive shells.

Troubleshooting

If you encounter any issues:

  • Ensure you meet all prerequisites.
  • Check if the environment variables SPARK_HOME and HADOOP_HOME are set correctly.
  • Review the script's output for any error messages.
